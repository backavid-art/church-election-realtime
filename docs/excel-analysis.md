# 엑셀 구조 분석 (원본: 2023 양평동교회 장로 1차 결과)

## 시트 구성
- 장로 1차 선거 개표 집계표: 최종 순위/득표율/당선
- 조별합계표: A/B/C조 합산표
- 개표집계표A: A조 입력 원본
- 개표집계표B: B조 입력 원본
- 개표집계표C: C조 입력 원본
- Sheet1: 보조/실험 데이터로 보이며 본집계 수식 참조 없음

## 열 구조

### 개표집계표A/B/C
- A 연번 (number)
- B 일렬번호 (number)
- C~E 1/2/3 선택 후보명 (text)
- F 비고 (text)
- H 후보합계 연번 (number)
- I 후보명 (text)
- J 후보별 COUNTIF 합계 (formula -> number)

팀별 하단 집계
- J66: 무효표 셀 카운트(빈 칸 개수)
  - A: `COUNTIF(($C$4:$E$111),"")`
  - B: `COUNTIF(($C$4:$E$153),"")`
  - C: `COUNTIF(($C$4:$E$154),"")`
- J67: `SUM(J5:J66)/3`

### 조별합계표
- A 기호(연번)
- B 후보명
- C A조 득표(`개표집계표A!Jx`)
- D B조 득표(`개표집계표B!Jx`)
- E C조 득표(`개표집계표C!Jx`)
- F 합계(`SUM(C:E)`)
- F61 유효투표 합계(팀별 유효표 합)
- F62 무효투표 합계(팀별 무효표 합)
- F63 총투표자 합계(팀별 총투표자 합)

### 장로 1차 선거 개표 집계표
- A 연번
- B 이름 (숨김영역 L:Q의 순위 lookup)
- C 총투표수 (=B4)
- D 득표수
- E 득표율 (=D/C)
- F 득표순위
- G 당선여부 (`IF(E>=0.666, "당선", "-")`)

숨김영역 L:Q
- L 득표순위: `RANK(P, $P$11:$P$60)`
- M 이름
- N 총투표수
- O 득표수
- P 득표율
- Q 당선여부

## 숨은 로직 해석
1. 득표율 분모는 유효투표가 아니라 `총투표자수(F63)`.
2. 당선 기준은 `득표율 >= 0.666` 고정.
3. 동점 처리 공식은 `RANK()`만 존재.
   - 공동순위 허용, 다음 순위는 건너뜀.
   - 별도 타이브레이커(연장자/재투표 등)는 수식에 없음.
4. 무효표는 최종표에서 별도 집계 칸(F62) 합산 방식.

## 이식 시 반영 포인트
- DB view에서 `vote_rate = votes / total_voters`로 계산.
- DB view에서 `rank() over (...)` 사용.
- 프론트에서 동점 공동순위 표시.
- 팀 입력폼에서 `유효표/무효표`를 별도 수치 입력하도록 구성.
